Como instalar Tailwind CSS CLI

Por favor ten en cuenta que estas usando la version CLI, que vendria siendo la que usamos en cualquier proyecto

> npm install tailwindcss @tailwindcss/cli



Importamos esta configuracion en el archivo main input.css

> @import "tailwindcss";



El archivo main de ingreso es input, pero el archivo main de salida, es como tu lo quieras llamar, para el caso practico se usa output.css, pero puede tener cualquier nombre y puede estar redirigiendo a cualquier ruta que se le especifique

> npx tailwindcss -i ./src/input.css -o ./src/output.css --watch



Finalmente llamas el archivo dentro del HTML y procedes con tu desarrollo

Si esto no funciona, siempre recuerda verificar la documentacion del paso a paso de instalacion aqui
